Acessibilidade

No topo das páginas de nosso site, disponibilizamos uma barra de acessibilidade com atalhos de teclado, facilitando a navegação para quem utiliza o teclado, como pessoas cegas e com certas limitações físicas.

Utilizando os atalhos

Em nosso site, disponibilizamos os seguintes atalhos de teclado:
1: Ir para o conteúdo principal da página
2: Ir para o menu principal
3: Abrir estas informações sobre acessibilidade
4: Alterar entre contraste normal e alto
5: Ir para página Mapa do Site

Cada navegador possui tecla ou teclas específicas para ativar os atalhos:
Chrome: Alt + [número do atalho] (Windows) ou Control + Option + [número do atalho] (Mac)
Firefox: Alt + Shift + [número do atalho] (Windows) ou Control + Option + [número do atalho] (Mac)
Internet Explorer: Alt + [número do atalho] (Windows) ou Control + [número do atalho] (Mac)
Safari: Alt + [número do atalho] (Windows) ou Control + Option + [número do atalho] (Mac)
Opera: Alt + [número do atalho]

Ampliando o texto no navegador

Pressione Ctrl + + (sinal de mais) para aumentar a fonte do texto;
Pressione Ctrl + - (sinal de menos) para diminuir a fonte do texto;
Pressione Ctrl + 0 (zero) para retornar ao tamanho padrão da fonte.

No Mac OS, substitua o Ctrl pela tecla Command.
